# Harris AI Enablement Candidate Campaign

Candidate campaign for Russell Dudek's application to the Harris AI Enablement Lead role.

## Campaign thesis

**Make local AI wins portable without centralizing local judgment.**

The campaign proposes a role-specific operating artifact: the **Mission Passport**, a lightweight transfer record for moving proven AI-enabled workflows across decentralized business units while preserving local ownership, trust decisions, and adaptation context.

## Files

- `index.html` - candidate vision site
- `resume.html` - role-aligned resume with exact two-page Letter print CSS
- `cover-letter.html` - exact one-page Letter print CSS
- `interview-brief.html` - mandate, tensions, thesis, proof map, objection, 90-day plan, scorecard, questions, and source list
- `mission-passport.html` - role-specific operating artifact for transferring proven AI-enabled workflows across decentralized business units
- `styles.css` - shared responsive and print styling
- `app.js` - accessible Mission Passport interaction
- `source-notes.md` - factual grounding and hypothesis boundaries
- `docs/` - validated printable PDFs

## Integrity note

The campaign separates public Harris facts from candidate hypotheses and uses verified candidate evidence. Company-specific operating recommendations are presented as hypotheses for discovery, not claims about undocumented internal process.
